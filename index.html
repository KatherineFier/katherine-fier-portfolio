Unplugged

HTML5, CSS3 & RWD from Photoshop design comp

Overview: Unplugged is a three page responsive website coded from a Photoshop Design Comp. The project included five phases: creating a project directory and repository; coding the HTML; coding the mobile, tablet, and desktop layout CSS; coding the general CSS for type, iframes, images and icons; and QAing the site.

Accelerate

PHP, HTML5, CSS3, RWD from Photoshop Design Comp

Overview: Accelerate is a five page responsive website. The project included coding a custom WordPress site from a Photoshop template. Guess the Word Game

JavaScript, HTML5, CSS3

Overview: The Guess the Word Game is a game where players can guess eight different letters as they try to figure out a random word that is pulled from an API.

The HTML5 and CSS3 were previously coded for this project. The logic for fetching the random word, accepting and validating the player guesses, displaying the guessed letters and word, calculating the remaining guesses, and the two buttons were needed.


Challenge: A key challenge with this project was getting the "Play it Again" button to function correctly. To address the issue, I studied variable scope and the key differences between using const and let. I then changed the "const guessedLetters" variable to "let guessedLetters," the playItAgainButton event handler functioned.


Tools: JavaScript, Arrays & Array Methods, Loops, Variables & Variable Scope, Functions, Async Function, Arguments, Parameters, Properties, Event Listeners & Handlers, Visual Studio Code, Chrome DevTools

Magic 8 Ball

JavaScript, jQuery, HTML5, CSS3

Overview: The Magic Eight Ball UI includes an "Ask Me Anything" button that enables users to click the button, ask a yes or no question, and recieve an answer. The HTML5 and CSS3 were previously coded for this project. The logic for the button, prompt window, and answers were needed.


Challenge: Figuring out how to randomly select values from the array of answers. I met this challenge by familiarizing myself with array methods and properties such as Math.random (), Math.floor (), and the length property.


Tools: Variables, Arrays, Methods, Properties, Event Listeners & Handlers, Git, Github, W3Schools, Chrome DevTools
