Boulder Flower Farm

Branding, Visual Design, Website Build, Digital Marketing

Overview: The Boulder Flower Farm was a five part project that included research and branding, visual design, a website build, marketing strategy and implementation, and customer experience.


Challenge: Getting stakeholder buy in to engage in digital marketing. To overcome this challenge, I proposed that I start small with digital marketing and track the results. Unplugged

HTML5, CSS3 & RWD from Photoshop design comp

Overview: Unplugged was a three page responsive website coded from a Photoshop Design Comp. The project included multiple phases including: creating a project directory and repository; coding the HTML; coding the mobile, tablet, and desktop layout CSS; coding the general CSS for type, iframes, images and icons; and QAing the site.


Challenge: Effectively using Flexbox to create the navigation and grid layouts. To overcome this challenge, I solidified my understanding of specific flexbox properties including: flex-flow, justify-content, and align-items.


Tools: HTML5, CSS3, Flexbox, Media Queries, Breakpoints, Responsive Typography, Font Awesome, Google Fonts, HTML5 Boilerplate, Git, GitHub, Adobe Photoshop, Chrome DevTools, W3Schools,

Magic Eight Ball

JavaScript & jQuery

Overview: The Magic Eight Ball UI included an "Ask Me Anything" button that enabled users to click the button, ask a yes or no question, and recieve an answer. The HTML5 and CSS3 were previously coded for this project. The logic for the button, prompt window, and answers were needed.


Challenge: Figuring out how to randomly select values from the array of answers. I met this challenge by familiarizing myself with array methods and properties such as Math.random (), Math.floor (), and the length property.


Tools: Variables, Arrays, Methods, Properties, Git, Github, W3Schools, Code Sandbox

Sticky Note UI

React, JSX, ES6

Overview: The Sticky Note UI allowed users to create new sticky notes, and then edit, delete and search the notes. This was a greenfield project that included three phases: building a component tree, managing the data flow and then making the components interactive.


Challenge: Setting up the data flow properly. Solidifying my understanding of container components versus presentational components and how to pass props enabled me to overcome this challenge and complete the project.


Tools: Props, Arrow Methods, Container and Presentational Components, Functional and Stateful Components, Code Sandbox
